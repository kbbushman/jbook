# Notepadjs

In-browser JavaScript editor with package import support. To get started, open your terminal and run:

```
$ npx notepadjs serve

```

### Technologies Used:

- Typescript
- Express
- React
- Redux
- Redux Thunk
- Immer
- ESBuild
- ESBuild-WASM
- Commander
- Lerna
- Monaco Editor
- Prettier
- Bulma
- LocalForage
